# JF News · João Fonseca

Site independente de fãs com as últimas notícias do João Fonseca, tenista brasileiro.

## Setup

1. Clone o repositório
2. Crie um arquivo `.env.local` com sua chave da API Anthropic:
   ```
   ANTHROPIC_API_KEY=sua-chave-aqui
   ```
3. `npm install`
4. `npm run dev`

## Deploy na Vercel

1. Importe o repositório do GitHub na Vercel
2. Adicione a variável de ambiente `ANTHROPIC_API_KEY` nas configurações
3. Deploy!
